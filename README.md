# Digital Forensics 101

This repository contains introductory slides for **Digital Forensics**, prepared as supporting material for the **Czechitas Digital Academy – Cybersecurity (Digitální akademie Kybernetická bezpečnost)**.

The slides are intended as **pre-workshop study material**. Participants are encouraged to go through them before attending the practical workshop.

The goal of this presentation is to introduce the basic concepts of:

- digital forensics
- forensic acquisition
- file system analysis
- recovery of deleted data
- timeline reconstruction
- common forensic tools
